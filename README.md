# Soundmode

Location-Based Sound Mode Automation is an Android app that switches sound profiles based on saved places.

## Run in Android Studio

1. Install Android Studio (latest stable) and the Android SDK.
2. Open Android Studio and select **Open**.
3. Choose the `Soundmode` repository folder.
4. Let Gradle sync finish.
5. Connect a device or start an emulator.
6. Click **Run â–¶** and select the `app` configuration.

## Build from the command line (optional)

```bash
./gradlew assembleDebug
```
